# Politique de confidentialité de SC Atlas

Dernière mise à jour : 28/04/2026

SC Atlas est un outil communautaire non officiel destiné aux joueurs francophones de Star Citizen. Cette politique explique quelles données sont utilisées par l’application et comment elles sont traitées.

## Données collectées

SC Atlas ne collecte pas de données personnelles et ne nécessite pas de compte utilisateur.

L’application ne transmet pas de données personnelles, de fichiers de jeu, de logs Star Citizen ou de statistiques d’utilisation à un serveur externe.

## Données stockées localement

SC Atlas peut stocker localement certains paramètres nécessaires à son fonctionnement, notamment :

- le dossier Star Citizen mémorisé ;
- les préférences de démarrage et de vérification des mises à jour ;
- l’état local de la traduction installée ;
- le résultat de l’analyse du temps estimé depuis les logs locaux.

Ces données restent sur l’ordinateur de l’utilisateur, dans le stockage local de l’application.

## Analyse des logs Star Citizen

SC Atlas peut analyser localement les fichiers logs de Star Citizen afin d’estimer un temps de jeu depuis les logs disponibles sur le PC.

Cette analyse est effectuée uniquement sur l’ordinateur de l’utilisateur. Les logs ne sont pas envoyés, copiés ou transmis à un serveur externe.

Le temps affiché est une estimation basée sur les fichiers présents localement. Il ne s’agit pas d’un temps de jeu officiel fourni par Cloud Imperium Games ou Roberts Space Industries.

## Connexions réseau

SC Atlas peut se connecter à des ressources publiques hébergées sur GitHub afin de :

- vérifier si une nouvelle version de SC Atlas est disponible ;
- vérifier si une nouvelle version de la traduction FR est disponible ;
- télécharger le fichier de traduction FR ;
- charger le bloc d’actualité affiché dans l’application.

SC Atlas peut aussi ouvrir des liens externes dans le navigateur de l’utilisateur, par exemple GitHub, Discord ou SCMDB FR.

## Services tiers

Les liens externes ouverts par SC Atlas peuvent être soumis aux politiques de confidentialité des services concernés, notamment GitHub, Discord ou SCMDB.

SC Atlas n’est pas responsable du contenu ni des pratiques de confidentialité de ces services tiers.

## Projet non officiel

SC Atlas n’est pas un outil officiel de Cloud Imperium Games, Roberts Space Industries ou Star Citizen.

L’application ne modifie pas le gameplay de Star Citizen.

## Contact

Pour signaler un problème ou poser une question, utilisez le dépôt GitHub du projet ou le serveur Discord de support indiqué dans l’application.
